# Lucia Auth Template

This repository provides a ready-to-use template for authentication using [Lucia](https://lucia-auth.com).

## Features

- ✅ GitHub OAuth
- ✅ Session Management
- 🛡️ Ready for 2FA/WebAuthn extensions
- 📁 Clean, modular code structure
